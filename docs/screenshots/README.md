# Screenshots Guide

This folder holds the images referenced by the main [`README.md`](../../README.md).
Capture the screens below and save them here using the **exact filenames** so the README
renders them automatically.

> **Tip:** Use a consistent browser width (~1440px) for public pages and full-window
> captures for the admin dashboard. PNG is preferred; keep each file under ~1 MB.

## Capturing tips

- **Hide personal/test data** you don't want public (real emails, addresses, keys).
- **Use seeded/test data** so the screens look populated and professional.
- For the **Stripe page**, use test card `4242 4242 4242 4242` and crop out any test-mode banners if you prefer a clean look (or keep them — they signal it's a safe demo).
- Optionally annotate one or two images (arrows/labels) to highlight standout features.

Once added, the image grid in the main `README.md` will display automatically — no further
edits needed.
